# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** sup@ragnos.io

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and provide a detailed response within 5 business days.

## Scope

This repository contains documentation and templates only (no executable code).
Security concerns are most likely to involve:

- Prompt injection vectors in template files
- Patterns that could lead to unsafe agent behavior
- Information disclosure in examples
